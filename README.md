# imgimg

Stitch two images side by side with optional captions

## Usage

~~~
Usage: ./imgimg.sh [OPTIONS] IMAGE_1 IMAGE2
Stitch two images and put captions below.

OPTIONS
  -l,  --labels     Add labels separated by comma. eg: before,after
~~~
